# Servers

> All .pem files check on official file sharing systems.

- `global-bundle.pem`: for database connection with SSL
- `adh-db-proxy.pem`: for Proxy connection server for MongoDB
- `adh-appserver.pem`: for logging into app server
- `adh-webserver.pem`: for logging into web server
- `adh-db.pem`: for logging into db server
